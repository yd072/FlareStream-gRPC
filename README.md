# FlareStream-gRPC 🚀

这是一个专为 Cloudflare Workers 编写的 VLESS over gRPC 代理节点。

## 🌟 核心特性

- **极致性能**：纯原生 JS 实现，零外部依赖，延迟极低。
- **智能回退**：自动无缝切换至备用 ProxyIP。
- **环境安全**：支持通过环境变量设置 UUID，无需在代码中明文硬编码。

## 🚀 快速部署

1. **创建 Worker**：在 Cloudflare 控制面板创建一个新的 Worker。（必须拥有域名，域名须在网络中开启`gRPC`允许通过 `gRPC` 连接到您的源服务器。不支持`Pages`部署）
2. **部署**：将本项目中的 `_worker.js` 内容复制并粘贴到编辑器中。
3. **设置环境变量**：
   - 进入 `设置 -> 变量和机密`。
   - 添加变量名 `UUID`，值为你的 UUID（例如：`d342d11e-d424-4583-b36e-524ab1f0afa4`）。
4. **保存并部署**。

## 🛠️ 客户端配置

- **协议 (Protocol)**: VLESS
- **地址**：你的域名 (或优选IP)
- **端口**：443
- **用户ID**：你的UUID
- **流控**：留空
- **加密**：none
- **传输协议**：grpc
- **伪装域名**：你的域名
- **路径 (ServiceName)**: 
  - 直连模式：`/` (或任意字符)
  - ProxyIP 模式：`/?ip=你的优选IP:端口` (Worker 会在直连失败时自动使用该 IP 降级)
- **传输安全**：tls
- **SNI**：你的域名
- **ALPN**：填写 h2 (非常重要，gRPC 强依赖 HTTP/2)
